# E-commerce Sales Performance Dashboard

## Overview

This project presents an interactive Power BI dashboard analyzing e-commerce sales data from 2014 to 2017. It highlights key business metrics, sales trends, and growth patterns.

## Key Features

* KPI tracking: Total Sales, Profit, Orders, and Profit Margin
* Monthly sales trend analysis
* Month-over-Month sales growth using DAX
* Category-wise performance analysis

## Tools Used

* Power BI
* DAX (Data Analysis Expressions)

## Key Insights

* Sales show a consistent upward trend over time
* Technology is the top-performing category
* Sales peak during year-end periods
* Growth fluctuates month-to-month, indicating demand variability
* Profit margin remains around 12%
